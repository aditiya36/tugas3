# Crypto Tracker

Aplikasi web mobile untuk menampilkan data harga cryptocurrency secara real-time menggunakan API Coinlore.

## Cara Menjalankan

```bash
npm install
npm run dev
```

Buka `http://localhost:4321/` di browser.
